Setting the Background

Two different methods

Option 1: feh

Install feh

Add one line to end of i3 configuration file

exec --no-startup-id feh --bg-scale ~/Pictures/cloudsAF1080.jpg

Option 2: xsetroot

Preferred by some over feh

Lighter install (xsetroot only does one thing)

Capability of setting background as solid color

exec --no-startup-id xsetroot -solid "#333333"

i3-gaps

Popular fork of i3

Provides gap configuration in windows

How to Install i3-gaps

Debian/Ubuntu/Other: Install from source

Arch: Install from AUR under i3-gaps package

Reference for all here

NOTE: It will install over an existing i3 install perfectly fine

How to Use i3-gaps

Use it just like normal i3 except...

WITH GAPS!

Gaps declared with pixel count on inner and outer spacings between windows/edge of screen

Pro Tips for i3-gaps

Generally windows look better without a border or title bar in i3-gaps

Set this with for_window [class="^.*"] border pixel 0

Pro Tips for i3-gaps/general i3 (continued)

Refreshing the layout of i3 is very useful

Do so after changing config with $mod+Shift+r or other set keybind
